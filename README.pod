=head1 ZedLog v0.1beta

ZedLog is a robust cross-platform input logging tool (AKA a key 
logger).  It is based on a flexible data logging system which makes it 
easy to get the required data.  The current features include the 
following;

=over 4

=item * All keyboard and mouse events.

=item * Full GUI

=item * Logging to file

=item * Basic hiding

=back

=head2 Getting Started

The ZedLog GUI is fairly intuitive but there is also full usage doc-
umentation under the help menu (or just press F1).
